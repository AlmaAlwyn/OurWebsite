# Microtech Beavers Website

Professional web development company website built with HTML, CSS, and JavaScript.

## Files Included

- `index.html` - Main HTML file
- `styles.css` - Stylesheet with custom color scheme
- `script.js` - JavaScript for interactivity
- `favicon.svg` - Website icon
- `hero-bg.svg` - Hero section background

## Hosting Instructions

### Option 1: Netlify (Recommended - Free & Easy)

1. Go to https://www.netlify.com and sign up
2. Click "Add new site" → "Deploy manually"
3. Drag and drop this entire folder
4. Your site will be live instantly!
5. Purchase domain at Namecheap/GoDaddy
6. Connect domain in Netlify settings

### Option 2: GitHub Pages (Free)

1. Create a GitHub account at https://github.com
2. Create a new repository named "microtechbeavers"
3. Upload all files to the repository
4. Go to Settings → Pages
5. Select "main" branch and save
6. Your site will be at: username.github.io/microtechbeavers
7. Purchase domain and configure DNS

### Option 3: Traditional Hosting (cPanel)

1. Purchase hosting from Hostinger/Bluehost/SiteGround
2. Get FTP credentials from hosting provider
3. Use FileZilla to upload files to public_html folder
4. Domain will be active within 24-48 hours
